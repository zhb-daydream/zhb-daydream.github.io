Typora双击放大预览图片(LightBox)

发表于2024-04-25|更新于2024-04-26|学术模块
|字数总计:960|阅读时长:3分钟

写在前面

用 Typora 写 markdown 的一大好处就是插图容易，直接粘贴图片就可以自动在指定路径里放置对应的图片了（当然，word 也是可以的，这里主要是和 vscode 编辑器比），但有一点不好，就是没法放大看细节。有了这个需求，就想去看看有没有类似的插件，然后就真的找到啦！（感谢大佬们）

基于 Lightbox 插件，可以实现以下两个功能

  • 双击放大图片，再双击关闭放大的图片
  • 可以使用滚轮对放大图片进行缩放

安装可以调试过程参考以下三个链接：

  • CSDN博客-Typora配置双击图片放大的功能(详细教程)
  • CSDN博客-Typora结合LightBox放大查看图片
  • 知乎专栏-Typora添加双击图片放大功能

下载 Lightbox

从 LightBox 的 Github 网址（https://github.com/lokesh/lightbox2）下载 dist 文件夹中的内容至 Typora 的[安装根目录]\resources\extend\lightbox2 中，如下图所示

修改 window.html 文件

window.html 文件是用来定义 Typora 的 UI 界面，其路径在 [根目录]\resources\window.html 中。其动态反馈基本按照前端的三件套（HTML、CSS、JavaScript）来定义，因而要增加“双击图片放大”的功能，也是得修改这个文件。

引入 css

在首行的 <head> xxx <\head> 内容末尾中，添加 lightbox 的 css 文件。由于文本内容比较多，可以直接搜索 <\head> 字符串，并在它的前面添加。

引入 js 文件

在 <script src="./appsrc/window/frame.js" defer="defer"></script> 的后面一行添加 lightbox 的 js 文件。

修改 lightbox.js 文件

lightbox 不是专门给 Typora 开发的，它默认的应用场景中需要有 <a> 标签包围在 <img> 标签外，同时点击按钮的事件也是绑定在 <a> 标签上的，而且还需要定义 href 、 data-lightbox 、 data-alt 等属性。由于 Typora 中的图片只有 <img> 标签，因此需要调整一下 lightbox 的文件定义，使其适配 Typora。这个调整就是修改 lightbox.js 文件，其路径位于前文中下载的 lightbox 文件夹里：[Typora 根目录]\resources\extend\lightbox2\js\lightbox.js
