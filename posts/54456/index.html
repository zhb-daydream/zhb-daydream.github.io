写在前面

突然想做个单独的图片展示区域，不仅是旅游图片，还包括一些游戏里面的梗图。而这些图片的展示，目前主要需求就是纯展示，并不需要专门为此写一大段文字去描述了，所以感觉一个专门的图片展示区（相册）是比较得劲的。

要注意的是，要在 Hexo 里面创建一个相册，主要是和 Hexo 主题绑定的，每个主题可能会有不同的创建方式，而且也需要（只需要）修改主题文件夹下的 _config.yml 配置文件。

首先百度了下 Hexo 怎么创建相册，发现还是有不少教程的，而且这个 liyang 的博主写了个很详细的教程，可复现程度也很高。但就如上面所说的，这个事情是和 Hexo 主题深度绑定的，所以最后还是决定直接去目前的主题（Butterfly）文档里面找教程（才不是因为这个博主写的方法太复杂而不想干呢😊 ）

创建相册主页面-Butterfly

Butterfly 主题是有现成的定义相册的方法的！在它的官方文档（文档-图库）中提到，在 Butterfly 配置中，相册主页与相册子页面其实都和 post 一样，是普通的页面，使用 .md 文件来定义。区别只在于需要使用标签外挂 galleryGroup 来定义里面的内容，而不是像正常的 Markdown 文本格式那样定义。

注意：在另一个官方文档（文档-相册图库）中还提到，标签外挂是 Hexo 特有的功能，而不是标准的 Markdown 格式；而下面的写法也仅适用于 Butterfly 主题，其他 Hexo 主题不一定能支持。
