Hexo更换主题

发表于2023-08-16|更新于2024-04-19|博客搭建
|字数总计:624|阅读时长:2分钟

虽然 NexT 6.0 用着挺好，很多功能我也还没试过怎么用，不过感觉还是到了尝试新主题的时候了（转折有点生硬），再加上之前一直都没有记录怎么安装 Hexo 的主题，也算是一种补完了吧。

Hexo 主题搜索

在 Hexo 的官网中，就收录了 388 个主题（https://hexo.io/themes/），含有该主题的一个小图以及作者的链接。不过一部分的链接以及无法打开了，这时候就得求助搜索引擎了。除了去 Hexo 官网去找心仪的主题外，还可以通过各种自媒体、文章、搜索引擎去找别人推荐的博客主题。对我这种小白来说，可能后者更适合吧，至少还有人帮忙总结要点。

我就不在这里瞎推荐了，目前为止只用过 NeXT，主要是更偏向于纯文本的网站，很简洁干净。不过最近想换个炫酷一点的主题，然后要有文章标题+左右侧边栏的结构。

Hexo 主题安装

一般的安装就是从 Github 下载并解压到 [Hexo根目录]\themes（或者通过命令行用 git 命令下载），然后把 Hexo 根目录的 _config.yml 配置文件中 Extensions 的 theme 后面改成对应主题文件夹名字就能生效了。

这里以 Butterfly 主题（演示界面：https://butterfly.js.org/）为例，首先去其 Github 仓库（https://github.com/jerryc127/hexo-theme-butterfly/tree/master）下载压缩包，master 分支会比 dev 分支更稳定，而后者可能会有一些新功能。

然后解压到 themes 文件夹并重命名（主要是强迫症，不想要那么长的前缀），接着修改 Hexo 根目录的配置文件 theme: butterfly。

最后，对于 Butterfly 主题来说，还需要两个 Hexo 渲染器插件 pug 与 stylus 才能正常编译。通过以下命令行去安装，如果安装进度条不动，多半是网络问题，挂个加速器之类的看看。

今天暂时就写这么多了，后续的排版、字数统计之类的功能就之后再弄吧。Butterfly 渲染的网页确实炫酷很多，但渲染时间也更长，之前的 NeXT 只要 2s，现在 Butterfly 要 10s。
