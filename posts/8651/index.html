MATLAB的两种GUI开发工具

GUIDE

图像用户界面开发环境（GUIDE，Graphical User Interface Development Environment）是 Matlab 早期版本（主要是 R2016a 之前的版本）中使用的一种 GUI 设计开发工具，MATLAB 在之后的版本中将会删除该功能，目前版本（R2020b）似乎只能在命令行窗口里通过输入 guide 来打开窗口，这里就不讲这个了。

APP designer

Matlab App Designer 是 Matlab R2016a 版本之后推出的新型 GUI 设计工具，界面更好看一点，底层函数的编写也更加模块化一些，而且基本是面向对象的应用，其 .mlpp 文件就是一个类 class，而不是脚本里的 function。通过在主页点击 APP-设计APP 或者在命令行输入 appdesigner 就可以打开设计工具。

mlapp文件结构

下面主要讲 Matlab App Designer 的内容。首先是该开发工具所对应的文件为 .mlapp 文件，其代码视图（在右上角中可以切换设计视图与代码视图）中主要有回调、函数、属性三个部分，这三个部分基本就是 UI 开发者所需要修改的所有东西了。

  • 回调（Recall）：与 UI 界面组件绑定的状态函数，根据 UI 交互状态进行传参、调用函数等操作
  • 函数（Function）：用户自定义的函数，用于计算并传参
  • 属性（Properties）：用于存储一些用户自定义的全局变量

一些基本功能

UI 用户交互界面的代码其实和普通的脚本、函数没什么不同，唯一多的东西就是前者还需要考虑交互变量与交互状态的传递，这个具体实现就是回调模块了。以下主要讲一些基本的操作。

按钮组件

按钮（button）组件感觉会是一个比较基础的组件，差不多可以用来当做代码执行的按钮。在设计视图下添加了该组件后，在组件浏览器里就会多出了一个 app.Button，右键并添加回调，就会在代码视图下看到增加了一个回调函数。按钮组件比较简单，只有一个状态：按下（Pushed），因而在回调函数里下的内容，就表示按下按钮后，程序会执行的内容，一般可以在里面加入绘图、加载文件等代码，这样就和 .m 脚本里的 F5 运行按钮一样了。
