LaTeX本地安装方法

发表于 2023-08-05 分类于 学术模块
本文字数： 1.1k 阅读时长 ≈ 4 分钟

LaTeX的官网是https://www.latex-project.org/，在另一篇博客《LaTeX与TeX的关系》也提到，LaTeX是使用TeX排版引擎生成PDF文件的一种语言，所以要使用LaTeX，就要有两个环境：（1）含有TeX引擎的TeX发行版，（2）一个LaTeX文本编辑器。

目前LaTeX可以在Linux、MacOS、Windows操作系统下部署并运行，也可以通过Overleaf、Papeeria、LaTeX base等在平台使用。对于在线使用而言，只需要一个浏览器进行在线的文本编辑就可以编译所需的PDF文件并下载，不需要额外安装TeX引擎并进行配置。但缺点也很明显，如果断网、网络不好，那用起来就很抓狂，所以这里我就只列LaTeX的本地（Windows）配置。

安装TeX排版引擎

对于Windows用户而言，可以无脑选择TeXLive，可以直接点开该TeXLive的官网链接，也可以在LaTeX官网中进行跳转。

下载并解压文件后，点击运行程序（一般是instal-tl-windows.bat），然后会在线下载、安装3G左右的内容，基本就是全自动完成的。

LaTeX编辑器-VSCode

LaTeX的编辑器我就只推荐VSCode了，颜值又高，功能又多（平常也是用这个IDE写python的），此外还有个TeXStudio，好像UI界面不太好看，而且我没用过。

VSCode（Visual Studio Code）的安装基本就是去官网下载安装包，然后一键安装。然后就是安装LaTeX的扩展插件，按如下图所示，只需要安装一个LaTeX Workshop的插件就能用了。

使用方法

打开一个.tex主文件（比如这里使用的是IEEE Access提供的模板文件），通过侧边菜单栏进行LaTeX文件的编译，即可在左下角看到Build。此外也可以直接使用保存的快捷键ctrl+s就能进行编译。

编译后，在.tex文件的同目录下就会出现编译好的PDF文件，可以直接在VSCode里面预览，点击右上角的预览按钮（也可以用侧边栏里面的View LaTeX PDF选项）。

然后就能愉快地边编辑、边预览编译结果了。
