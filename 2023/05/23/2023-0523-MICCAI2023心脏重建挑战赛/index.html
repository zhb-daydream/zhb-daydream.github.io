MICCAI2023心脏重建挑战赛

发表于 2023-05-23 更新于 2023-06-26 分类于 MRI
本文字数： 1.5k 阅读时长 ≈ 5 分钟

背景

MICCAI2023 CMRxRecon(https://cmrxrecon.github.io)心脏快速成像挑战赛

数据集:我们将首次公开300人的心脏多通道kspace数据集，包括短轴、2腔、3腔、4腔长轴电影和T1/T2 mapping，约500,000张图像。

挑战任务:本次挑战赛包含两个子任务，即高倍欠采样的cine重建和mapping重建，挑战的加速倍数分别为4倍、8倍和10倍。 奖金:

本次挑战赛由西门子独家冠名赞助，每个子任务的第一名可获得1000美金，第二名将获得500美金，第三名获得300美金! 本次CMRxRecon挑战赛由复旦大学、帝国理工大学、香港理工大学、厦门大学、德克萨斯大学等10所单位联合组织，详情请点击: https://mrxrecon.github.io。

注册流程

官方主页（github）：https://cmrxrecon.github.io

Synapse 主页（用于下载数据、提交代码）：https://www.synapse.org/#!Synapse:syn51471091/wiki/622170

  • Synapse 是一个合作性的开源学术平台，用于各 team 进行数据分享、分析和讨论。当一些大学联合举办某个挑战赛时，就会在 Synapse 上建立一个 Team，报名者以各自的 Synapse 账号为单位参加。
  • 本次挑战赛在 Synapse 的 Team 名字为 CMRxRecon Participants
  • 需要往 CMRxRecon@outlook.com 邮箱发送 RegistrationForm 和 CMRxReconDataAccessAgreement，里面包含参赛组名（自己编）、组员、所属单位、Synapse 账号（用于提交代码和下载数据），邮箱格式如下。

  • 发送邮箱后等待 Synapse 的 CMRxRecon 组通过即可
  • 注册的 Synapse 账号还需要升级到 Certified User （需要在个人主页完成一些问答题）才能进行代码上传、数据下载

数据下载

成功注册进 CMRxRecon Challenge 后（会收到对应邮件），可以从挑战赛说明网页 的最下方进入数据下载界面，包含 120 例 training data，60 例 validation data，120 例 test data，各例数据中又含有加速倍率为 4/8/10 的数据及欠采模板。

重建 demo 代码

在 CMRxRecon 的 github 主页 可以找到开源的代码（不需要注册 Synapse 账号），主要有几个部分

  • CMRxReconDemo: contains parallel imaging reconstruction code
  • ChallengeDataFormat: Explain the challenge data and the rules for data submission
  • Evaluation: contains image quality evaluation code for validation and testing (from the FastMRI project: https://github.com/facebookresearch/fastMRI/)
  • Mapping: contains fitting code for T1 mapping and T2 mapping
  • Submission: contains the structure for challenge submission
