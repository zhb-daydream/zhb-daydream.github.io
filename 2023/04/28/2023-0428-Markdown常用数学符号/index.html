最开始是想在高铁上想抄点数学公式，结果碰到些生僻符号，又加上一直穿隧道，信号极差，想了想还是得自己整理个常用的 markdown 符号才行，万事不求人嘛。手敲是不可能手敲的，这种表格肯定得用图片的，而且也不太想用别人的带水印图片。

在 Markdown 官网 处有作者 Matt Cone 的比较全的说明书《The Markdown Guide》（可惜要 5 美元），似乎也有对应的简体中文版（CSDN 上的一众博客大概都是用的这本书的截图），万能的 gen.lib.rus.ec 上能搜到相应的资源，可惜高铁上一直加载不出来。

在必应上搜索的时候，在 www.cta.org 上找到个 Scott Pakin 在 2021 年 5 月写的《The Comprehensive LaTex Symbol List》的完整资源，可惜这本书写得也没 Markdown 的那个那么简洁。

最后还是从 某个不知道从哪跳出来的网页 里找到无水印版的部分简体中文版 Markdown 指南，所以就直接盗用了！
