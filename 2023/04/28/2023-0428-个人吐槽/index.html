2023-0428个人吐槽

发表于 2023-04-28 更新于 2023-06-01 分类于 个人吐槽
本文字数： 550 阅读时长 ≈ 2 分钟

在高铁上，信号极差，顿时感觉人类社会脱离了网络、脱离了信息社区，会是一个多么可怕的事情。本来想整理完 LORAKS 的代码之后补个 stablediffusion 的软件安装教程，结果中途写着写着发现有些生僻数学符号不会，查的时候又加载半天，就想去整理个 Markdown 的常用数学符号文章，然后找资源的路上又是加载半天（头秃）。最后一顿抄完 LORAKS 的公式，也没加深理解多少，感觉这个就只能当成是一个兴趣爱好来做了，不然 “为了学习 LORAKS 而敲了几小时键盘，结果啥也没学到”这种事情说出去，多少会给整破防的。

一看时间也错过饭点了（大意了，应该买点面包吃的），打算弄完这部分就尝试看点之前缓存的 ISMRM education talk。上传的时候用之前的 python 脚本给所有 .md 文件检查了一遍相对路径，结果它全给我修改了，本来残次不齐的修改时间，愣是给弄的整整齐齐，只能改了下，只在检测到不符合 Hexo 要求的 \ref_pic\ 时才进行文本替换。还是得风格各异才舒服呀，前篇一律、整整齐齐的风格只能用在非主观的事情上呢。

唉，今天就这样吧，没有搜索引擎，好多事都效率特别低，完善博客内容、框架这种事情也只能以后再做了。（此处应有表情包，但是网络信号实在太卡了）
