2023-0424日记

发表于 2023-04-24 更新于 2023-06-28 分类于 日记
本文字数： 408 阅读时长 ≈ 1 分钟

唉，一天又过去惹，早上想摸鱼整理了个《如何确定研究领域大牛》的文章，本来都还挺顺利，到一点半就弄完了的，结果编译的时候一直卡在 “图片相对路径错误”、“公式编译错误”两个地方，而且还是偶发性的。最后检查发现就是 hexo-renderer-marked 的问题，本来指望这个渲染器来处理前者的图片路径问题，结果它本身又会导致公式编译出错。

最后呢，卸载了 hexo-renderer-marked，既然不想更改 Typora 的使用习惯（使用 ./ref_pic 或者 /ref_pic 引用），又没法改 Hexo 的逻辑（只能识别 /ref_pic），那就只能加个批处理文件，修改一下 .md 里面的文本咯。

现在暂时是通过 python 脚本实现（python 还是方便呀！可惜我不会用），感觉是可以用 js 实现的（毕竟 Hexo 根目录里面还有个 node_modules 文件夹呢），但。。那也以后再说吧
