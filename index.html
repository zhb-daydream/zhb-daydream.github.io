如何正确使用拐杖

发表于 2023-06-17 分类于 搞点课外知识
本文字数： 993 阅读时长 ≈ 3 分钟

（注：致我的崴到脚的女朋友。）

拐杖定义

拐杖是—种辅助行走的简单器械（通常是一根木制或金属棍子）也是一种重要的医疗康复辅助用具。一般有以下三种分类：

  • 手杖：主要用于轻度需要例如老年人或登山者，不属于残疾人用品
  • 肘杖：肘杖属于中度下肢残疾人用品
  • 腋杖：腋杖是下肢重度残疾者（病残伤残等）的必需品，一般是左右各一个

拐杖挑选

负重程度

使用拐杖的目的就是减轻或避免下肢负重、帮助维持身体平衡。而根据患者症状，医生一般会划分为五种负重程度（即使用拐杖时要达到的目的）：

  • 不负重（Non-weightbearing，NWB）∶即患腿不受力，也就是保持你的患腿离开地面
  • 轻负重（Feather-weightbearing，FeWB）∶可以用脚趾点地来维持平衡
  • 部分负重（Partial-weightbearing，PWB）∶可以将身体部分体重分担到患腿上
  • 可忍耐负重（Weightbearing as tolerated，WBAT）∶将大部分体重甚至所有重量负担到患脚，能忍耐即可
  • 全负重（Full-weightbearing，FWB）∶只要患肢不痛，可完全受力

单拐放置位置

使用单拐时，单拐的放置位置会有患侧、健侧两种选择：

  • 对于截肢患者：单拐应在患侧，用拐杖代替患肢实现行走功能
  • 对于不能完全负重患者：单拐应在健侧，用于将重心放在健侧，从而减少患侧的负重与身体横向移动，增加身体稳定性

单拐步态

平地行走

  1. 正常站立保持稳定
  2. 重心转移到健侧
  3. 患腿与拐杖同时向前迈（向前距离一致，横向宽度要给健侧腿留有空间）
  4. 重心转移到患腿和拐杖
  5. 健侧腿向前迈
  6. 回到第 2 步

上楼梯

  1. 健侧腿迈上台阶
  2. 重心转移到健侧腿
  3. 拐杖与患腿迈上台阶
  4. 回到第 1 步

下楼梯

  1. 重心在健侧腿上
  2. 拐杖与患腿迈到下一个台阶
  3. 重心移到拐杖与患腿上
  4. 迈出健侧腿
  5. 回到第 1 步

==注意==：用握住拐杖手柄来支撑你的体重，不是用腋窝顶在拐杖上。因腋窝有重要的血管神经丛通过，以免受压损伤

专利申请流程

发表于 2023-06-16 分类于 搞点课外知识
本文字数： 1.3k 阅读时长 ≈ 4 分钟

专利类型

  • 发明专利：针对产品、方法或者产品、方法的改进所提出的新的技术方案。
  • 实用新型专利：针对产品的形状、构造或者其结合所提出的适于实用的新的技术方案。
  • 外观设计专利：针对产品的形状、图案或者其结合以及色彩与形状、图案的结合所作出的富有美感并适于工业应用的新设计。

涉及的人员类型

  • 发明人：即作者（第一发明人对应于论文的一作）。必须是法人，一篇专利可有多个发明人。
  • 申请人：拥有专利权利的所有人及持有人（专利授权前的称呼）。也称专利申请权人，包括自然人、各单位、院校、企业。
  • 专利权人：拥有专利权利的所有人及持有人（专利授权后的称呼）。包括自然人、各单位、院校、企业。
  • 代理人：受申请人委托，为其办理专利申请、转让、许可等事务的代办人

专利申请流程

对于发明专利而言：

  1. 提出申请
  2. 专利局受理
  3. 初步审查
  4. 公布阶段
  5. 实质审查（含第一次审查意见等）
  6. 授权公告

对于实用新型专利和外观设计专利而言：

  1. 提出申请
  2. 专利局受理
  3. 初步审查
  4. 授权公告

审查周期

普通申请通道审查周期 优先审查通道审查周期 快速预审通道审查周期
发明 18-26个月 7-12个月 3-6个月
实用新型 7-8个月 3-6个月 1-2个月
外观设计 3-6个月 3-6个月 5-7个工作日

优先审查通道需要满⾜以下条件（主要是对领域的要求）：

  1. 涉及国家大力扶持的领域（如新材料、新能源、生物医药、节能环保、智能制造、互联网大数据等）。
  2. 属于各省级、市级重点扶持的领域。
  3. 申请人已经做好专利内容的实施准备或者已经开始实施。

快速预审通道需要满⾜以下条件：

  1. 权利人的注册或登记地在保护中⼼所属⾏政区域，且为具有独⽴法⼈资格的企业、高校或科研院所；
  2. 权利人的⽣产、研发或经营⽅向，属于保护中⼼的快速预审领域，即高端装备制造产业或新材料产业；
  3. 权利人没有非正常专利申请、故意侵犯他人知识产权等不良记录。

如何查阅专利

发表于 2023-06-16 分类于 搞点课外知识
本文字数： 870 阅读时长 ≈ 3 分钟

一篇专利，除了其正文内容比较重要以外，它的申请人、发明人、申请时间、法律状态等信息也同样重要，因此一个方便好用（甚至免费）的查询渠道就比较关键。

推荐平台

专利汇⭐

网址是：www.patenthub.cn（看起来和代码管理的 github、文献下载的 scihub 一样，都有个 hub，比较好记）。

很推荐该平台，主打一个免费方便，能满足大部分的使用需求。

  • 注册即用
  • 响应速度（比官方的查阅系统）快
  • PDF 文件下载无限制
  • 数据更新快（能查到一篇 10 天前刚授权的专利）
  • 筛选功能全

知嘟嘟

网址是：https://www.iprdb.com/。

推荐作为专利汇的备选，注册即用，响应速度很快。

政府平台（不太好用）

中国专利公布公告

网址是：http://epub.cnipa.gov.cn/，属于国家知识产权局下的政府网站，主办单位：国家知识产权局　软件维护：知识产权出版社。

不推荐使用，免注册、检索速度慢（一分钟过去可能还是啥也没有）。放这里纯是当一个反面例子。

中国及多国专利审查信息查询

网址是：https://link.zhihu.com/?target=http%3A//cpquery.cnipa.gov.cn/，属于国家知识产权局下的政府网站。

不推荐使用，全面权威，但响应速度慢（时不时就加载不出来）。

专利检索及分析

网址是：https://pss-system.cponline.cnipa.gov.cn/conventionalSearch

不推荐使用，注册即可免费使用，但响应速度慢。

其他平台

谷歌专利

网址是：https://patents.google.com/。

可以检索到各专利信息，也可以免费下载，但检索结果的统计信息似乎要比专利汇的少一点。

高校

一般各高校图书馆网址里的电子资源等栏目里，都有对应的专利搜索引擎入口，通过高校游客账号访问即可。
