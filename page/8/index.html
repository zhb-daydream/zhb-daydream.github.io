结构化矩阵

发表于 2023-04-23 更新于 2023-06-01 分类于 学术模块
本文字数： 1.2k 阅读时长 ≈ 4 分钟

结构化矩阵（Structured Matrix）是指可以用少于矩阵元素数量的参数去描述的矩阵，即如果一个 阶的矩阵只需要少于 个参数来描述，那这个矩阵就是结构化矩阵。通过“可以用更少的参量去描述该矩阵”这一特点，就能减少矩阵计算时的内存消耗，并利用快速的矩阵-向量乘法和梯度计算方法来极大地提高计算效率，当然，相应的代价便是近似时可能会引入一定偏差。

常见的结构化矩阵有 Hankel 矩阵、Toeplitz 矩阵：

（1）Toeplitz 矩阵（同一条对角线上的元素相等，2n-1 个独立元素） （2）循环矩阵（circulant matrix，特殊的 Toeplitz 矩阵） （3）Hankel 矩阵（同一条反对角线上元素相等）

发表于 2023-04-21 更新于 2023-04-28 分类于 博客搭建

Hexo 的静态网页是通过 Markdown 格式的文本文件生成的，而在 Markdown 文件中插入图片的语法为 ![图片描述](图片路径)，其中图片路径可以分为以下三种。

  • 网络路径：以网址形式存储，十分方便，但源图片失效或网站限制（如微信公众号）时会无法加载图片，且访问速度较慢
  • 绝对路径：在计算机的绝对位置，不适用于博客的部署
  • 相对路径：相对于当前文件的路径

若要使用图片相对路径，则需要进行相应的配置，因为 Hexo 并不支持相对路径的引用。

根据习惯调整

上述存储方式实际上是与我自己的写作习惯冲突的：

  • 个人写作习惯：在 Typora 直接粘贴图片，图片默认保存路径为 ref_pic/imag.jpg 或 ./ref_pic/imag.jpg（后者需要将 Typora 升级到 1.4.3 之后的版本）
  • Hexo 图片索引规则：以 source 为根目录进行索引，且只能识别以斜杠 / 开头的相对路径。

接下来的改进需求是：

  • 核心需求：不改变以往的图片导入习惯，让 Hexo 能自动识别无 / 开头的图片相对路径
  • 次要需求：只维护一个图片保存文件夹，且该文件夹位置按照以往习惯保存在 .md 文件同目录下（即 \source\_post\ 下）

目前（2023-0423）只能满足核心需求：同时维护 \source\ref_pic 和 \source\_post\ref_pic 两个文件夹，前者用于 Hexo 识别，后者为了符合 Typora 使用习惯。此外引入 python 脚本，自动将 ](/ref_pic/ 或 ](/ref_pic 替换成 ](/ref_pic，需要每次 Hexo 编译前运行一次脚本。

奇异值与特征值

发表于 2023-04-21 更新于 2023-04-28 分类于 学术模块
本文字数： 1.5k 阅读时长 ≈ 5 分钟

参考知乎回答：https://zhuanlan.zhihu.com/p/353637184

奇异值（Singular Value）与特征值（Eigen Value）是两个数据分析中重要又相近的概念。早在 1907 年的时候，德国数学家 Erhard Schmidt（矩阵求逆、标准正交基中的 Gram-Schmidt 方法）等人提出了奇异值的概念，但当时的奇异值被称为"eigenvalues"，即特征值，直到 1937 年才由 F.Smithies 开始使用奇异值"Singular Value"的术语。

目前的大数据时代，SVD（Singular Value Decomposition，奇异值分解）是最重要的矩阵分解方法，广泛用于矩阵的低秩近似、伪逆计算、PCA（Principal Component Analysis，主成分分析）等场景中，而且一般情况下 SVD 都很 robust，能确保解的存在。

相应的，EVD（Eigen Value Decomposition，特征值分解）就不能保证一定存在解，但特征值也很重要，因为它体现了矩阵内在的特性，用于薛定谔方程中的能级表征、马尔科夫均衡态计算、微分方程中的相图边界划分、谱聚类中的谱特征值。要注意的是，只有方阵才有特征值，而实际应用中的矩阵 大都不是方阵，因而一般会通过做左自乘： 得到性质优良的对称矩阵 ，再进行后续的 EVD 计算，而自乘得到的对称矩阵 也被 Gilbert Strang 称为线性代数的皇帝。

概念与定义

奇异值与特征值都被用于描述矩阵作用在向量上时的效果，都是定量描述向量模长变化幅度的标量、数值，它们的差异在于：

  • 特征向量描述的是矩阵的方向不变作用（Invariant Action）的向量
  • 奇异向量描述的是矩阵的最大作用（Maximum Action）的方向向量

这里的作用（action）意思是：当矩阵与向量乘积得到了一个新的向量后，该过程在几何上等价为对向量进行了旋转和拉伸，该变换过程就是矩阵对向量所施加的作用。

  • 若有向量 与矩阵 之间的矩阵-向量积满足 （ 为标量），则 和 就是分别是矩阵 的特征值和特征向量，此时若将所有特征值构造成对角阵 、归一化的特征向量（列向量）构成正交矩阵 ，则矩阵 可以写作：
  • 若存在单位正交矩阵 和 ，使得矩阵 满足 （ 为对角矩阵），则 对角线上的值被称为矩阵 的奇异值， 和 中的列分别被称为矩阵 的左奇异向量和右奇异向量

关联：

当矩阵 为方阵时，它同时具有奇异值和特征值，两者存在一定的关联：

  • 若 为对称方阵，则它的奇异值等于特征值的绝对值，对应的左右奇异向量与特征向量平行/共线
  • 若 为一般方阵，则它的奇异值等于 和 的特征值的平方根，左奇异向量 为右自乘 的特征向量，右奇异向量 为左自乘 的特征向量。

  • 将 js 文件尽可能放置到 body 的闭合标签之前
