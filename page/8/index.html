比赛介绍

第二届中国科学技术大学量化交易研究大赛由信宏天禾与中科大联合举办，将提供真实的金融量化业务痛点，为希望从事量化交易相关工作的同学提供真实而有深度的实战模拟。推荐组合：金融+物理或数学+计算机或机械自动化等工科。

  • 一等奖1组：3万奖金及证书
  • 二等奖1组：1万奖金及证书
  • 三等奖2组：5千奖金及证书
  • 参与奖（提交成果即可）：证书

信宏天禾背景

官网：http://x-asset.com/，创立于2012年10月，在2015年取得私募基金管理人资质，2021年入驻北京基金小镇，具备投顾资质。主要开展资产管理业务，以数理统计、深度学习为理论基础，专注于全球资本市场策略研发，策略覆盖量化选股、指数增强、中性对冲、CTA、FOF等策略。 办公地点在上海、北京。

端午游-扬州-下

发表于 2023-06-25 更新于 2023-06-27 分类于 生活版块
本文字数： 819 阅读时长 ≈ 3 分钟

博物馆

上午点了个生煎锅贴王的外卖，外卖小哥送到前台后，前台安排了个小机器人一路送到房门口，还是有点高级的。点了4个生煎包、2个煎饺、1份馄饨、1份元宵，共47.1，量大好吃，甚至点多了还剩不少。打了会塞尔达，然后12点退房开溜。

免费预约了个12点-4点的大运河博物馆，好像还有个古风系列的密室逃脱门票可以抢，但基本没票，在每天凌晨12点会放出7天后的预约，也就是说想去的话，就得提前7天蹲点去守着了，对不做计划的人极不友好。

虽然是淡季，但博物馆有好多旅行团，一个是老年旅行团，另一个就是小学生旅行团，前者就是慢慢走、看得特别仔细，后者则是去明显是给小孩子准备的湿地动物主题馆里抄写简介。由于人太多，展品也很多，花了20大洋租的语音解说器基本派不上用场，单是过个眼瘾就已经走得疲惫不堪了。

瘦西湖

逛完博物馆，找地方买了个扬大牛奶，也算是扬州特别出名的酸奶了，又便宜又好喝，只出没于平民日常生活的区域，无法在大商圈里找到。

既然来了扬州，那瘦西湖总得去一下吧！门票是60元/人（白天，下午7点离开）、120元/人（夜晚，有灯光秀），我们大方针就是去西门停车场停好车，然后就溜达一小块，感受一下就溜。

瘦西湖里的风景都很丰富，移步换景，过了一座小桥便是新的风景，而且不同分叉口似乎都会通往不同的目的地，最后就大概地逛了二十四桥、白塔，看了小鸽子、小鸭子、小天鹅子。整体上还是很惬意的，走走停停，比博物馆惬意多了。

返程

想尝一下著名的金陵烤鸭，随便找个商圈（万象汇）去吃个吉祥春饼.烤鸭，不用排队，但总感觉好像和青年公社的北京烤鸭没什么区别，也有可能是走累了。

吃完就是开夜车回合肥了。不得不说，开高速困的时候，还是需要有人在旁边一直逼叨逼叨聊天才能保持清醒。期间有个小闸口，交警会拦车检查行驶证和驾照，似乎是专门拦的外地车，不知道是不是在抓逃犯呢。

回到学校，去到熟悉的西区东门宵夜摊，吃完之后就算是圆满结束这次4天3夜的旅途啦！

端午游-扬州-上

发表于 2023-06-24 更新于 2023-06-27 分类于 生活版块
本文字数： 1.1k 阅读时长 ≈ 4 分钟

出发

中午去到连云港海州区的万象汇城，大中午出门真的热死个人。逛了一圈，决定就吃个麦当劳解决了。原定是今天（6.24）结束旅程回去的，但总有点点意犹未尽，经过一番讨论，决定先绕去扬州再待一天！（其实就是加完油之后临时起意的）

要开高速，自然也少不了先买一杯瑞幸咖啡啦（真的没有打广告），喝着冰的、开着车内演唱会，还是较为舒适的，而且天气也很不错，借着天窗可以赏云。

开到大概下午6点半的时候，天开始变黑，还穿过了几片乌云。一共开了大概3个半小时，在傍晚7点23的时候到达扬州迎宾馆，下着雨进酒店。

这个扬州迎宾馆，很不一般，它不是大不大的问题，它真的是那种，那种很少见的那种（当然也可能是我们见识少了）。总体而言，整个酒店就是一个大园区、住宅区，小区绿化覆盖率极高，颇有进了园林的感觉，园区大到设有24小时循环运行、接送的电瓶车，其中的6号楼是独栋别墅，6888元/栋（共5间房）。我们订的是APP上的3号楼的498元标准双床房（无窗），由于是6.24周六已经是端午节最后一天，属于淡季，还免费升级到了价值568元的带窗房间。

办好入住后，前台遥控一按，地上还有指引射灯来引导电梯位置，进房间之后还送了若干粽子和扬州零食，免费送虹桥坊温泉两张入门票（价值19×2=38元），甚至浴室有浴缸，总之就是一路刘姥姥进大观园既视感。

入住瘫了一会之后准备找吃的，发现想点的生煎包外卖已经没了，只能去罗森买点零食，不过听说虹桥坊温泉里有吃的，而且开到凌晨2点，就坐园区小电瓶车过去了。

泡澡

虹桥坊温泉其实就是虹桥坊酒店的一个下属设施，酒店住客可以免费进去一次，其余人也可以在App上买票进入。门票（也称浴资）本身就是19元/人，会分配一个手环，进去之后的所有消费项目，包括吃饭（没有免费自助餐！）、擦背、按脚、SPA等都会先记账到手环上，在最后出去时一起结账。而酒店赠送的门票/浴资就是个二维码代金券，在消费时出示即可。

我进去之后才搞懂，原来泡澡就是特指男汤、女汤那里的冲澡，而这里是要一丝不挂的！虽然听说各种小温泉都泡得很舒服，但我在一堆赤身裸体里面总感觉特别特别不自在，低着头走路也不是，昂着头走路也不是，最关键的是要脱光衣服进去冲了个澡才能换套浴衣。虽说身体发肤受之父母，还都是哥们，不必有什么害怕的，但我还是红着脸逃出来了。（P.S. 还有另一种泡澡称私汤，这种就是私人泡的，不必与陌生人坦诚相见）

男汤、女汤以外的区域是可以自由活动的，除了上述收费项目，也有免费的设施，比如儿童用的波波池（当然，我肯定也去体验了一下）、观影室、汗蒸等等，但我吃了个32块钱的扬州炒饭之后就开始觉得无聊了。看来这种温泉，精髓还是在那个男汤、女汤区域啊。

端午游-连云港-下

发表于 2023-06-23 更新于 2023-06-27 分类于 生活版块
本文字数： 1.5k 阅读时长 ≈ 5 分钟

赖床

13点起床之后吃了点东西，因为下午的阳光太热了，完全不想离开空调，就主要在玩手机+王国之泪，然后查攻略下一站要去哪。如果要查看一个地方的商圈在哪，除了翻小红书等攻略论坛，最直接的就是去看电影院都在哪。基本确定了连云区就两个大商圈，一个是嘉瑞宝商业广场，另一个是栖霞路步行街（也叫墟沟步行街），不过这两地基本就距离一个block。

等4点半之后出门，把两个地方都大概逛了下，整体感觉就是比较复古（？）的小县城商圈，莫名有种老家的亲切感。补充了点零食，买了蜜雪冰城、意大利雪糕（Gelato）、烧饼、西瓜，然后继续去下一站了。其实栖霞路边上还有个左心房鲜果茶饮挺出名，但走的时候没有查位置，最后也懒得去了。

这个《小心玻璃》的背后仿佛有什么故事。

日落

下午6点半去到连岛准备看日落，停好车后刚好赶上连岛风景区里的接送大巴往外送人，逐渐变得堵车了起来。先去游客中心瞄了下，才算是搞清楚连岛风景区大概是什么回事，在白天营业期间，游客车辆只能停在游客中心，并通过大巴摆渡车进景区（要收费），而在夜间就可以自行开车进去，只不过这时候也没啥景色看就是了。

看日落，那必然挨着海，但西大堤又没有人行道和楼梯，怎么办呢？那当然是模仿其他大哥，翻围栏、穿马路、爬大堤栏杆了！大堤栏杆还是有点难度的（如下图所示），实心墙面有大概1.5米高，得先抓住上方的栏杆，然后蹬着墙往上，最后再双腿跨过栏杆才到。

虽然爬的时候擦破了点膝盖，但视野很不错，感觉很赚！一直目送着太阳远去，一如早上迎接它的降临一般。

边上还有一只小昆虫（鄙人文盲，实在不知道是什么了），静静的看着落日方向，不知道是睡着了还是已经嘎了。可能短短的一天就已经是它的一生了吧，就像《逍遥游.北冥有鱼》里面说的“朝菌不知晦朔，蟪蛄不知春秋”一般。

兜风

由于晚上7点左右是游客集中离开的高峰期，再加上我自己也比较拖拉、懒惰，就一直在车里待到了8点半才继续出发去兜风。一开始是抱着在海风下面踩踩沙滩的目标开进连岛的，虽然游客中心出来的路上显示只能左转出去，但还是硬着头皮右拐进连岛景区了。

连岛景区很大，里面除了各种风景区，也有大量的酒店、本地人居住区、餐饮饭店等。按进岛时候的环岛分叉口，连岛大概有三个区域：

  • 西连岛：各种海鲜饭店、大排档，本地人的住宅以及低档点的酒店，晚上灯会稍微多一点，没景点看
  • 山上：高档点的酒店和饭店，晚上山路又黑又绕，而且没东西看，大概是适合早上在房间看日出吧
  • 东连岛：区域很大，只是开车逛了一小段，各种收费的景区和设施，最东边的苏马湾就是用来看日出的，但那是走栈道过去，并没有沙滩可以逛

在东连岛散步的时候还碰到一家人，女的抱着小孩骂另一个男的，男方的爸妈就站在一旁不说话，偷听到的就是“错过了最后两班车，这么晚了还怎么出去？”，“出来玩都不听我的意见，那要我干什么？就是帮你们带孩子呗”，猜测可能就是人太多，然后老人家又不想做死贵的黑车，就这么僵持在这里了，大概之后只能就近找更贵的住宿了吧。

海边沙滩

9点半出连岛，10点到了连云港在海一方公园，这个可能是附近唯一的免费、公开、设施完善的海边沙滩公园了吧。在马路边斥5块巨资买了个小桶和小铲子，去沙滩上愉快地掺沙子、堆城堡（海拉鲁城堡！）、挖螃蟹，确实就是简单的快乐呢。

P.S. 穿着拖鞋沿海走的时候，一定要记得脚趾用力抓着

离开的时候一脚的沙，边上的冲脚机居然还收2块钱一分钟，硬是在公园边上找了个厕所去洗沙子。虽然边上明确贴着《不许洗脚》，但从一地的泥沙也可以看出来并没有什么用，而且我洗的是鞋子不是脚，也不算坏了规矩对吧😊。

晚上11点42回到宾馆，结束这天行程。

端午游-连云港-上

发表于 2023-06-22 更新于 2023-06-27 分类于 生活版块
本文字数： 1.8k 阅读时长 ≈ 6 分钟

出发

女朋友硕士顺利毕业啦！在变成社畜之前赶紧去自驾游一下转转！要论自驾游，怎么能少了全场 9.9 的瑞幸咖啡呢？13:40搞了杯冷的丝滑拿铁就出发！

本次目的地连云港市（苏G）主要分为海州区（连云港站）和连云区（连云港东站）两部分，前者就是看个桃花涧和花果山，后者才是看海所要去的地方。一路上听着公路歌单，换手开了5小时，终于在19:25的时候到连云港连云区。刚好路上欣赏了个落日，沿着花果山、云台山左绕右绕，怀揣着对大海的向往稀里糊涂地进了连云区，发现啥也看不到（要看海还是得去景区呢）

抵达

晚上吃的新尚粤皇港式茶餐厅，三份菜（炒河粉、炸酸奶、双拼腊肠饭）共94，不过分量确实很足，大概是因为这店就是为旁边的小区服务的，所以不会太宰客？不得不说，旁边的帝皇水榭花都小区看上去真高档，查了下房价，居然只有7k-9k/平？

由于还是端午节，行程订的又仓促，导致只能订个又贵又偏的酒店（为我的变卦和不靠谱而道歉）。住的地方叫芳芳宾馆（大概是为了一位叫芳芳的女孩子吧），432两晚，其实就是一个超市（好朋友超市）的二层改造而成的，由超市老板统一管理。虽然就在连云港东站附近（那里一堆酒店），但出奇的安静，而且到目前为止也没有堵车，可能这就是海边小城人民独有的休闲感？（也有可能是因为还没到连岛之类的核心景区区域）。总之就是入住然后休息，准备第二天看日出（刚开了王国之泪的新档，真好玩）

日出

照理来说，连云港主要就是连岛一个景区值得玩的（海边），去看日出也应该都去那里挤。但是，在宾馆老板的推荐下，我们还是去了高公岛看日出，其实和连岛差不多也是个最东的地方，由于要自驾一大段距离，人估计也没有连岛多，而且还能经过一段长达5公里的田湾跨海大桥，感觉是个值得去的地方。

定了第二天6.23凌晨3点闹钟，3点15眯着眼挣扎着起来，3点40正式开车出发。这里还是辛苦老板帮忙开门了，因为这家超市是男主人和女主人轮流值班的，女日班、男日班，晚上的时候男主人就在一楼收银台后面的床上睡，把通往二楼宾馆区的门锁上，我们凌晨要出去就得敲门叫醒他来开门（老板还是很和善的）。

沿着云宿路一直开，虽然没车，但一堆红路灯，一开始跟着个大哥，跟丢了。后面有另一个大哥，每个红绿灯都恰到好处地经过，不急不慢、不浪费一滴油。

经过了跨海大桥，开窗吹着风，还是没有什么车的宽阔三车道，看着蒙蒙亮的天空，还是很不错的。唯一有点遗憾的就是海风味道不弄，感觉就是普通的风。

在高公岛停车场停好车、沿着小路一直走，然后经过一段礁石，听着女朋友介绍的地质知识，找了块宽敞的石头坐下来。人不少，但也在可接受范围内。一直待到5:50左右就开溜，期间拍了不少照片，但能出片的却几乎没有，看来我得找时间稍微学一下怎么拍照了😢。

兜风

早上6点的时候，天就基本全亮了，想着半夜经过田湾跨海大桥的时候都没停下来看看，有点可惜又折返回去，在两侧的临时停靠点待了一会，这时候带的折叠椅就派上用场了！咔咔一顿拍照，好像还不错（如果不是公开博客，就在这里放合照了）。

说起来，高公岛附近还有个田湾核电站，居然还是中俄合建的（1999年开建1号机组，2018年4号机组投入商用），在桥上就能看到4个火电站那种大圆柱，地图上也是核电命名的建筑和路。

沿着连高路和海滨大道一路北上兜风，两车道，慢慢开，慢慢看，很舒适，看到边上一大片晒着渔网，还在想到时候怎么区分谁是谁的呢？渔网上会贴标签吗？

期间穿过了个3.6公里长的海上云台山隧道，比较老，建于1984年，双向两车道，但里面特别宽阔，通风很好，很凉爽。看着这庞大的人防工程，有种奇妙的感觉，大自然固然有鬼斧神工，但人类文明也能建造出庞大的工业产品，一时之间也找不到什么词汇描述，总之就是很微妙了。

穿过隧道发现里面其实是海上云台山的景区，在上高架回宾馆和原路折返继续沿海边兜风之间选择了后者。路上看到了一大片卸货码头，真的一眼望不到头（小小的中国震撼）。路上还有些奇奇怪怪的想吐槽的地名，比如《中山中路》、《连云港港》。

最后还绕去连岛景区的游客中心踩了个点。这里有很多黑车拉客进连岛景区的，而官方人员则立了个大牌子，语重心长地强调《拉客必宰客》，颇有种长辈的劝诫感觉。有趣的是，在连云区内，也有些标语是劝人不要吸食罂粟，其语气之慈祥，让我们有点摸不着头脑。

最后实在顶不住，大概8点多就回到宾馆了，吃点蛋黄派什么的就睡了，大概一觉睡到13点。
