sci论文的利益冲突是什么

发表于 2023-07-17 分类于 学术模块
本文字数： 757 阅读时长 ≈ 3 分钟

在sci论文投稿的时候，一般都会要求在正文或者cover letter中描述利益冲突（conflict of interest），而大部分人都是直接写 "We have no conflicts of interest to disclose"。那么，这个利益冲突具体是什么东西呢？

利益冲突的定义

利益冲突（conflict of interest）是指个人或组织的利益可能会影响论文客观性的情况。因为在论文写作中，难免会牵扯到资金支持、学术合作等，如果在撰写论文时资金资助来源较为复杂，而其中某项资金来源同其他相关研究有一定的关系，那么其研究数据乃至整个论文的客观性就会受到怀疑。这一类的潜在利益如果不能在科研发表工作时列出来，则读者可能会觉得自己被误导或欺骗，降低了科研过程的透明度以及期刊的可信度。因而为了防患于未然、保证学术研究的公正性和客观性，编辑会要求作者在投稿时准备好利益冲突声明。

利益冲突的类型

常见的利益冲突类型有两类：

  • 财务关系：最常见的利益冲突。如雇主、股东、赞助商支持、旅费津贴、专利等，是最有可能破坏期刊、作者甚至科学本身的信任的因素。为此，一般要说明资助或津贴的确切金额、对研究起到的作用和影响
  • 非财务关系：包括个人的家人/合作伙伴、学术竞争关系、意识形态与信仰等。需要阐明这些关系具体是什么，以及可能会引发怎样的潜在利益冲突。

为什么U盘比光盘更容易泄密

发表于 2023-07-16 更新于 2023-07-17 分类于 搞点课外知识
本文字数： 1.5k 阅读时长 ≈ 5 分钟

一般而言，涉密计算机可以用光盘，而不能用U盘，因为后者更容易植入、传播计算机病毒。

U盘与USB

U盘是USB盘（USB flash disk）的简称，它不需要硬盘那样的物理驱动器，可以通过USB接口实现即插即用，同时存储容量也远超软盘，极便于携带。总结而言就是一种使用USB接口的、无须物理驱动器的、微型高容量移动存储产品。

USB（通用串行总线，Universal Serial Bus）是一种串口总线标准，也是一种输入/输出接口的技术规范，在1994年底由英特尔等多家公司联合推出USB 1.0协议，传输速度为1.5Mb/s=0.1875MB/s。在2000年推出的USB 2.0的接口颜色是黑白，采用4针脚设计，传输速度为480Mb/s=60MB/s。在2009年推出的USB 3.0 的接口颜色是蓝色，采用9针脚设计，传输速度为4.8Gb/s=600MB/s。

USB设备风险

回到正文，U盘这一类使用USB协议的USB设备都有很大的泄密风险，主要是因为USB设备可以玩的花样太多了，可以通过内部编程把一个USB口的设备改造成具有各种功能的设备，包括：

  • 鼠标：只要一直让鼠标发送1像素的移动，几秒钟发一个，就可以确保电脑一直不进入屏保，除非是主动按快捷键进入锁屏状态。而只要不进入屏保，那么键盘就有了很大的发挥空间。
  • 键盘：可以发送个组合键Win+R，然后发送cmd和回车，就会得到个控制台，这个过程完全不需要看到屏幕的当前状态就可以实现。再之后无论是新建BAT批处理文件，执行硬盘搜索，可以做无数操作。
  • 摄像头：可以让U盘壳子的某个角落开个小孔放入USB摄像头，结合如上的键盘操作，就可以拍摄周围的环境。
  • 大容量存储：就是常见的U盘，不仅可以存储用户的数据，也可以自己隐藏的存储一些数据，还能模拟出来多个U盘。
  • 串口：可以使得电脑与外部以串口通信。
  • 声卡：结合键盘命令可以劫持电脑的输出音频。
  • 网卡：可以劫持网络通信。

相比之下，光盘就要简单的多，只是一个单纯的存储介质，无法进行编程、实现各种计算机功能。

USB数据线泄密案例

Linus Tech Tips在2023-0510发布过一个视频，讲述一个伪装成USB线的黑客设备（不过造价也高，大概200美元）。外表看上去就是一根普通的lighting充电线，但在X光照射下，就可以看到其复杂的内部结构。

一旦将这根数据线插入到电脑上，它就会模拟成一对键鼠，检测到计算机闲置一段时间后，就能操纵鼠标打开恶意链接，下载并打开对应的勒索或病毒软件。如果是在无网络状态下，也可以通过极快的速度打开命令行，以高达每分钟一千个单次的输入速度将预先设置好的恶意代码注入机器，或者粗暴地输入万恶的删库代码：rm -rf /*。可以在几分钟内通过该USB数据线将计算机攻陷，成为一台肉鸡。

更进一步地，该数据线还集成了一个WiFi模块，在成功注意恶意代码后，创建一个绕过计算机防火墙的WiFi连接，将计算机所有的重要数据传输出去，或者由攻击者本人进行远程操控。

此类USB设备可以是常见的U盘，也可以是数据线、转换头，还可以是键盘、鼠标、无线网卡，甚至是网络接口、HDMI接口。因而有个笑话：任何高明的黑客手段，都比不上花钱收买对方公司的保洁，趁半夜把服务器网线拔了。

MATLAB设置代码背景

发表于 2023-07-12 分类于 学术模块
本文字数： 282 阅读时长 ≈ 1 分钟

一个良好的代码编写环境能有效提高工作效率，舒服一点的代码背景能降低眼睛负担，更适合长时间工作。

下文中所有的背景颜色都是在 MATLAB 上方菜单栏的“主页”——“预设”处设置。

代码节高亮

在代码中，通过 %%（有空格）可以对代码进行分块/节，可以在显示的时候有明显的区域划分，也可以在代码执行的时候以节为单位进行。

在预设项中的“颜色”——“编程工具”——“节显示选项”中，可以关闭“突出显示节”，关闭代码分块的高亮。

代码背景

在预设项中的“颜色”可设置统一的背景（包括编辑器、命令行窗口、工作区等区域），这里可以把默认的白色设置一个柔和点的淡绿色。

sci如何写cover letter

本文字数： 262 阅读时长 ≈ 1 分钟

sci投稿时，有些期刊会要求上传一封给编辑的 Cover Letter（一般是用 Word 写），这种投稿附言信没有严格的格式要求，只要把主要信息都写到就行了。以下是泰勒&弗朗西斯（Taylor & Francis）出版社给出的投稿附言模板，可以参考其结构和句式。

段落的构成大致为：

  1. 本论文题目为【xxx】，准备投稿到【xxx】期刊
  2. 论文的原创性声明以及非一稿多投声明
  3. 本文的大致内容以及创新性
  4. 本文是如何符合期刊的 Aims & Scope 的
  5. 没有利益冲突
  6. 有问题就通过【xx】邮箱联系

MRI射频放大器供应商

本文字数： 370 阅读时长 ≈ 1 分钟

射频放大器

磁共振成像系统中的射频功率放大器（RF amplifier）主要由射频线圈、射频发生器、射频放大器组成，主要的作用是发射能够激发成像区域的射频脉冲，会对谱仪的输入 RF 波形（控制信号，电流较小）放大后，再输出到射频线圈中。

全球供应商

通过 everythingRF 可以进行检索（该网页也是通过在 google 上检索 mri rf amplifier 找到的），在筛选项中设置 Configuration=System、Type=Power Amplifier，可见基本只有澳大利亚的 Tomco、美国的 Communication Power Corporation 两家公司有丰富的产品线（该网站统计的不一定准确）。除此之外，一些磁共振设备供应商如宁波的鑫高益、沈阳的东软，一般是整机出售，不会单独进行部件的售卖。

LaTeX使用bib文件编译参考文献

发表于 2023-07-09 分类于 学术模块
本文字数： 1.3k 阅读时长 ≈ 4 分钟

涉及的文件

.tex文件

最常见的LaTeX文件，也是编写文章的文件。

.bib文件

BiBTeX文件，是一种用于管理参考文献的工具，将参考文献以一种标准化的格式存储（包括参考文献的作者、标题、出版社、出版日期等详细信息），并在LaTeX文档（.tex文件）中引用。每个参考文献都有一个唯一的标识符，即“键”，可以在.tex文件中使用这个键来引用该参考文献。例如，在正文段使用\cite{key}就会在文档中生成一个带有作者、出版日期的引用标记（具体的引用样式会在.tex文件中定义）

.bst文件

定义参考文献的引用样式的文件，一般在.tex文件中通过\bibliographystyle{xxx.bst}导入

.bbl文件

由BiBTeX根据某一个参考文献引用样式（.bst文件）生成的参考文献记录文件，可以将里面的内容复制粘贴到.tex文件中，通过\begin{thebibliography}{} \end{thebibliography}调用。

从EndNote导出bib文件

在EndNote中选中待导出的参考文献，依次点击左上角的 File — Export，在 Output style 中选择 BibTeX Export，输出为 .txt 文件。

随后直接修改文件后缀名为 .bib 即可。通过文本编辑器打开可见 .bib 文件中列出了某篇文献的所有标准化信息，该信息是通用的，与参考样式无关。同时在红框处（RN508）还有各文献在该 .bib 文件中的唯一标识符（key），该标识符可以手动修改，在 .tex 文件的正文段中通过 \cite{RN508} 来引用文献。

在.tex中导入.bst文件和.bib文件

一般各个期刊提供的LaTeX模板文件中，都会包含定义了参考文献样式的.bst文件（例如 Medical Physics 期刊用的是 medphy.bst，IEEE Access 期刊用的是 IEEEtran.bst），如果没有的话就去搜索引擎下载一个。

将.bst文件放到.tex文件的同目录下，然后在.tex文件接近结束的地方通过以下两行来导入.bst文件和.bib文件（这里导入的是medphy.bst文件和ref.bib文件）

编译得到.bbl文件并替换.bib（可选）

在 VS Code 中，ctrl+s 保存后，就会自动进行 LaTeX 编译，该过程会输出 .bbl 文件。用文本编辑器打开，可见其内包含了指定参考文献引用样式下的文本信息。有些期刊会要求将.bbl这里的文本信息全都复制到.tex文件中，这样在后续编译的时候就不需要依赖.bib文件和.bst文件了。

打开.bbl文件并将其中的内容全部复制粘贴到.tex文件的参考文献区域中，注释掉或删除 \bibliography{references.bib} 命令。然后重新编译.tex文件，此时输出的PDF文件应与以前相同。
