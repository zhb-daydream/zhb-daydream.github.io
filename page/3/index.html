MATLAB设置代码背景

发表于 2023-07-12 分类于 学术模块
本文字数： 282 阅读时长 ≈ 1 分钟

一个良好的代码编写环境能有效提高工作效率，舒服一点的代码背景能降低眼睛负担，更适合长时间工作。

下文中所有的背景颜色都是在 MATLAB 上方菜单栏的“主页”——“预设”处设置。

代码节高亮

在代码中，通过 %%（有空格）可以对代码进行分块/节，可以在显示的时候有明显的区域划分，也可以在代码执行的时候以节为单位进行。

在预设项中的“颜色”——“编程工具”——“节显示选项”中，可以关闭“突出显示节”，关闭代码分块的高亮。

代码背景

在预设项中的“颜色”可设置统一的背景（包括编辑器、命令行窗口、工作区等区域），这里可以把默认的白色设置一个柔和点的淡绿色。

sci如何写cover letter

发表于 2023-07-10 更新于 2023-07-17 分类于 搞点课外知识
本文字数： 262 阅读时长 ≈ 1 分钟

sci投稿时，有些期刊会要求上传一封给编辑的 Cover Letter（一般是用 Word 写），这种投稿附言信没有严格的格式要求，只要把主要信息都写到就行了。以下是泰勒&弗朗西斯（Taylor & Francis）出版社给出的投稿附言模板，可以参考其结构和句式。

段落的构成大致为：

  1. 本论文题目为【xxx】，准备投稿到【xxx】期刊
  2. 论文的原创性声明以及非一稿多投声明
  3. 本文的大致内容以及创新性
  4. 本文是如何符合期刊的 Aims & Scope 的
  5. 没有利益冲突
  6. 有问题就通过【xx】邮箱联系

MRI射频放大器供应商

发表于 2023-07-10 分类于 MRI
本文字数： 370 阅读时长 ≈ 1 分钟

射频放大器

磁共振成像系统中的射频功率放大器（RF amplifier）主要由射频线圈、射频发生器、射频放大器组成，主要的作用是发射能够激发成像区域的射频脉冲，会对谱仪的输入 RF 波形（控制信号，电流较小）放大后，再输出到射频线圈中。

全球供应商

通过 everythingRF 可以进行检索（该网页也是通过在 google 上检索 mri rf amplifier 找到的），在筛选项中设置 Configuration=System、Type=Power Amplifier，可见基本只有澳大利亚的 Tomco、美国的 Communication Power Corporation 两家公司有丰富的产品线（该网站统计的不一定准确）。除此之外，一些磁共振设备供应商如宁波的鑫高益、沈阳的东软，一般是整机出售，不会单独进行部件的售卖。

LaTeX使用bib文件编译参考文献

发表于 2023-07-09 分类于 学术模块
本文字数： 1.3k 阅读时长 ≈ 4 分钟

涉及的文件

.tex文件

最常见的LaTeX文件，也是编写文章的文件。

.bib文件

BiBTeX文件，是一种用于管理参考文献的工具，将参考文献以一种标准化的格式存储（包括参考文献的作者、标题、出版社、出版日期等详细信息），并在LaTeX文档（.tex文件）中引用。每个参考文献都有一个唯一的标识符，即“键”，可以在.tex文件中使用这个键来引用该参考文献。例如，在正文段使用\cite{key}就会在文档中生成一个带有作者、出版日期的引用标记（具体的引用样式会在.tex文件中定义）

.bst文件

定义参考文献的引用样式的文件，一般在.tex文件中通过\bibliographystyle{xxx.bst}导入

.bbl文件

由BiBTeX根据某一个参考文献引用样式（.bst文件）生成的参考文献记录文件，可以将里面的内容复制粘贴到.tex文件中，通过\begin{thebibliography}{} \end{thebibliography}调用。

从EndNote导出bib文件

在EndNote中选中待导出的参考文献，依次点击左上角的 File — Export，在 Output style 中选择 BibTeX Export，输出为 .txt 文件。

随后直接修改文件后缀名为 .bib 即可。通过文本编辑器打开可见 .bib 文件中列出了某篇文献的所有标准化信息，该信息是通用的，与参考样式无关。同时在红框处（RN508）还有各文献在该 .bib 文件中的唯一标识符（key），该标识符可以手动修改，在 .tex 文件的正文段中通过 \cite{RN508} 来引用文献。

在.tex中导入.bst文件和.bib文件

一般各个期刊提供的LaTeX模板文件中，都会包含定义了参考文献样式的.bst文件（例如 Medical Physics 期刊用的是 medphy.bst，IEEE Access 期刊用的是 IEEEtran.bst），如果没有的话就去搜索引擎下载一个。

将.bst文件放到.tex文件的同目录下，然后在.tex文件接近结束的地方通过以下两行来导入.bst文件和.bib文件（这里导入的是medphy.bst文件和ref.bib文件）

编译得到.bbl文件并替换.bib（可选）

在 VS Code 中，ctrl+s 保存后，就会自动进行 LaTeX 编译，该过程会输出 .bbl 文件。用文本编辑器打开，可见其内包含了指定参考文献引用样式下的文本信息。有些期刊会要求将.bbl这里的文本信息全都复制到.tex文件中，这样在后续编译的时候就不需要依赖.bib文件和.bst文件了。

打开.bbl文件并将其中的内容全部复制粘贴到.tex文件的参考文献区域中，注释掉或删除 \bibliography{references.bib} 命令。然后重新编译.tex文件，此时输出的PDF文件应与以前相同。

LaTeX生成术语表

发表于 2023-07-09 分类于 学术模块
本文字数： 849 阅读时长 ≈ 3 分钟

在使用 LaTeX 写学术论文的时候，有些期刊会要求提供缩写词汇对应表（List of Abbreviations）、术语表（List of Terminology），一般就是通过调用一些库函数来实现的。

三种package

常用的库函数有 Nomenclature：\usepackage{nomencl}、Glossaries：\usepackage{glossaries}，但这两种库函数在线上的 LaTeX 编辑器（如Overleaf）中不需要额外的操作，但在本地编译的时候，需要通过 cmd 进行额外的编译操作（如 makeindex、makeglossaries），而不能直接在 VS Code 中保存-自动编译-预览。为了在本地编译的时候减少步骤，这里使用了UIUC的一个 Acro 库：\usepackage{acro}。

Acro使用方法

使用方法和选项可以参考官方文档：https://ctan.math.illinois.edu/macros/latex/contrib/acro/acro-manual.pdf

ChatGPT与NewBing的AI聊天引擎使用方法

发表于 2023-07-08 分类于 搞点课外知识
本文字数： 1.9k 阅读时长 ≈ 6 分钟
产品名 开发公司 推出时间 LLM
ChatGPT3.5 OpenAI 2022-11 GPT-3.5
ChatGPT4 OpenAI 2023-03 GPT-4
NewBing 微软(Microsoft) 2023-05 GPT-4
文心一言 百度 2023-03 Ernie
通义千问 阿里巴巴 2023-04 通义
Google bard 谷歌(Google) 2023-05 LaMDA

ChatGPT

首先，要使用 ChatGPT，就需要有 OpenAI 的账号（注册需要使用境外手机号进行验证，相关方法可以参考网上攻略）。在使用时，可以通过网页 chat.openai.com/chat 进行访问（需要大陆地区以外的IP才能访问），也可以用 OpenAI 的账户申请一个 OpenAI API Key 用于本地部署 OpenAI 相关的应用。

网页对话

访问网页：https://chat.openai.com/chat ，登陆账号（注意网络IP），然后就可以进入界面使用了。如果登陆不成功或提示失败（在2023-05那段时间经常出现访问不了的情况），就只能多试几次，或者换个时间段再试。

虽然交互界面中都是英文，但实际上 ChatGPT 是支持中文输入和输出的。一个账号可以在多个设备登录，但同时只能进行一段对话。目前网页版是基础版的 ChatGPT，搭载的是 GPT-3.5 模型，可免费使用，但对话结果是预训练好的、不支持联网搜索。如果升级到 ChatGPT Plus（每月$20）则可以使用可联网版的 GPT-4 模型。（但那样其实还不如直接使用同样搭载了 GPT-4 的免费 NewBing）

API Key

该方法用于开发基于 ChatGPT 的应用。首先需要获取 API（Application Programming Interface，应用程序编程接口）密钥，可以参考 获取 API Key 的教程。此处的 API Key 是用于外部访问的一个密钥，在 OpenAI 官网 登录账户后进行申请，以随机字符串形式保存，若丢失该字符串，需要再次生成。

OpenAI 提供的 API Key 是要收费的，因为调用 GPT 模型也是需要消耗计算机算力、电费的，在使用量巨大的时候该成本更是明显。1000个 token/令牌大概 0.002 刀，一般情况下有 18 美元的免费额度。其中，token 是 GPT 一类应用对文本进行划分的单位，1个 token 大约为 0.75 个词（word）。

正因为 API Key 是收费的，因而在2023年初的时候，大佬们开发出来一堆基于ChatGPT开发的免费应用（比如ChatGPT Next Web）也慢慢地开始要收费或者不好用了（当然，也可能是因为我很久没碰了，所以换地儿了）。之后有时间或许可以写一下基于ChatGPT的其他应用，比如文献分析之类的，毕竟AI时代，还是得学点相关的。

New Bing

在2023年5月4日，微软就宣布搭载了GPT模型的NewBing聊天机器人全面开放给所有用户，不再需要加入候补名单，只要有一个微软账户，且使用 Microsoft Edge 浏览器访问 www.bing.com，就可以立刻进行体验。（注：Firefox浏览器似乎不支持，而且需要非中国地区的IP）

访问bing主页、登录了微软账号、更换了IP地址后，就能在网页最上方看到“聊天”标签。

点开后，就能愉快地使用了！由于使用的是可联网的GPT-4，因而对话信息里会有实时的搜索信息，而且引用材料还挺全。目前支持30条的连续对话，感觉比ChatGPT好用。

其他

百度的文心一言、阿里的通义千问在使用的时候，也是差不多，而且由于是国内，不需要考虑ChatGPT那种使用国际手机注册、NewBing需要更换网络IP地址的额外步骤，这里就不讲它们的使用方法了。（主要是平常NewBing就够我用了，好像也暂时用不到其他的）
